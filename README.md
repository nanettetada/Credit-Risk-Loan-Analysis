#  Credit Risk & Loan Default Prediction
*A Data Science project focused on Financial Risk Management.*

## Project Overview
This project aims to help banks like First Capital Group minimize financial loss by predicting the likelihood of a borrower defaulting on a loan. Using a dataset of over 30k loan applications, I built a model to identify high-risk profiles.

##  Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn
* **Model:** Random Forest Classifier (chosen for high explainability in banking)
